## 10. Summary

- **Shared:** Header (Admin Panel, Administrator, Sign out), sidebar (all nav items), common (Total, Items per page, Showing…, View, Delete, Actions, Status, Email).
- **Profile:** Title, language/name sections, account info labels.
- **Gallery:** Title, descriptions, stats, filter label, category list (14 categories), card labels (View, Hide, Show, Delete, Active, Inactive).
- **Customer Uploads:** Title, description, note, filters, table headers, Read/Unread.
- **Approvals:** Title, description, auto-note, filters, table headers, Pending/Approved, Approval date, Auto-approve.
- **Audit Logs:** Title, description, auto-note, stats, filters, table headers, Not read yet.
- **Tracking:** Title, description, auto-note, stats, filters, table headers, Read At, Not read yet.
- **Customers:** Title, description, New Customer, filter, search placeholder, table headers, project/projects.
- **Folders:** Standard folder display names used in tables (Photos, Documents, Other, During Work, Before, After, Damages and Defects).

Use this list to add or align keys in `locales/en/common.json` and `locales/de/common.json`; many of these keys already exist from earlier translation work.
